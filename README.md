신경공학 팀플

1. 5분정도 되는 특정 영상을 틀어놓기.

2. 사람이 뇌파장치를 착용하고 위 영상을 관찰한다.

3.1. 뇌파 신호를 처리할 수 있는 데이터 형태로 변형
3.2 5분 영상이 끝나면 착용자의 뇌파를 분석한 데이터를 직관하기 쉽게 그래프로 보여주기.
3.3 위 3.1번을 자동화하기.

4.1 영상을 보는 과정 중에서 "집중"하고 있다고 파악되는 부분이 있을 것임.
4.2 (추가하면 좋을 듯) 사람 본인의 얼굴이 나오는 화면과 그 옆에 영상이 보이는 화면 추가.

5. 4번에서 파악한 부분을 자동으로 요약해주는 것(지능형 AI or 타임대별 중요 내용을 수동으로 코드 기입)이 최종 목적임.
